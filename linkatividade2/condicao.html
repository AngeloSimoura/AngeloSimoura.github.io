Condição
Chamamos de estruturas condicionais as instruções para testar se uma condição é verdadeira ou não. Elas podem ser associadas às estruturas que se repetem, após o cumprimento da condição, chamamos a elas de estruturas de loop.

Operadores Relacionais e Lógicos 

(3 > 2) é uma expressão lógica que possui valor verdadeiro. Por outro lado, (4 < 1) é uma expressão lógica que possui valor falso.
Os operadores (< e >) são chamados de operadores relacionais pois possibilitam saber qual a relação existente entre seus dois operandos.
Além destes dois existem mais 4 operadores relacionais:

Forma Significado
== Igual a
!= Diferente
>= Maior ou igual
> Maior que
< Menor que
<= Menor ou igual


Operadores Lógicos:
Forma Significado
|| OU lógico
%% E lógico
! Negação


	
Estes operadores OU e E devem ser sempre usados entre duas expressões relacionais, como por exemplo:

((2 > 1) || (3 < 7))  : resultado VERDADEIRO
((3 < 2) && (2 == 2)) : resultado FALSO
((5 !=0) || (1 < 2))  : resultado VERDADEIRO

O operador de Negação deve ser usado antes de uma expressão relacional, como por exemplo:

!(2 > 1): resultado VERDADEIRO
!(1 < 0): resultado FALSO

Comandos IF , ELSE , ELSE IF 

O comando IF serve para alterar o fluxo de execução de um programa em C baseado no valor, verdadeiro ou falso, de uma expressão lógica;
o comando ELSE IF só é acionado se: O IF anterior for falso;
O comando ELSE só é acionado se: o IF anterior for falso e se todos os comandos 'ELSE IF' forem falsos.

EXEMPLO:

main(){                                                         
	if(condicao){ // Executado se  a condição for verdadeira
		< conjunto de comandos 1 >                      
	}                                                       
                                                                
	else{ //Senão (condição 1 é falsa) execute o else       
		< conjunto de comandos 2 >                      
	}                                                       
}
